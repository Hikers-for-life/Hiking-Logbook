# Running the Project Locally

---

## Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Firebase project credentials

---

## Setup

1. Clone the repo:
   ```bash
   git clone <repo-url>
   cd Hiking-Logbook
   ```

Install dependencies:

npm --prefix frontend install
npm --prefix backend install

Add .env files with Firebase credentials:

/frontend/.env
